In this code I have performed a number of operations using a pipeline function. This function is applied on each dataset.

Sentiment Analysis of Tweets

Sentiment Analysis is a technique in Natural language processing where you can find out the emotion behind any text. It is very import for analyzing and visualizing data. I perform sentiment analysis here on every tweet of every file and classify the tweets as Positive, Neutral and Negative.

Count Vectorizer

Count Vectorizer is used to transform a given text into a vector on the basis of the frequency (count) of each word that occurs in the entire text. I have used Count Vectorizer on all the tweets and computed the 20 most common words along with their count of every team.
